# Neuroitch

Offline idea machine for Android.

**[Download the APK](https://github.com/rahulwalia94-jpg/neuroitch-apk/raw/main/neuroitch-release.apk)** (~49 MB)

After downloading on an Android phone: tap the file, allow installs from
this source when prompted, then Install.

`cards.json` is the live card pack - the installed app checks it at launch
and picks up new cards automatically.
